# 软件入门基础套件指南

首先不得不介绍的是软件开发必备的一些基础工具。在有经验的情况下，它们使用起来可能非常简单自然，但是如果你是第一次接触，可能会对于如何建立一个基础的网站，完成一个简单的软件开发项目感到非常困惑。

我将从我自己的经验出发，分享一些简单易上手的基础开发工具流，让你明白如何实现一个可用的软件项目。
